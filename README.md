# 🌡️ Conversor de Temperatura

Um projeto simples de **conversão de temperatura** desenvolvido para praticar **HTML, CSS e JavaScript**.
A aplicação permite converter valores entre diferentes escalas de temperatura de forma rápida e intuitiva.

## 🚀 Demonstração

Você pode acessar o projeto online através do GitHub Pages:

https://andre-studionerd.github.io/conversor_temperatura/

## 📋 Funcionalidades

* Conversão de temperatura entre diferentes unidades
* Interface simples e fácil de usar
* Atualização rápida do resultado
* Projeto leve e totalmente executado no navegador

Conversores normalmente trabalham entre escalas como **Celsius, Fahrenheit e Kelvin**, que são as principais unidades usadas para medir temperatura no mundo. ([ichi.pro][1])

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 💻 Como executar o projeto

1. Clone o repositório

git clone https://github.com/Andre-StudioNerd/conversor_temperatura.git

2. Acesse a pasta do projeto

cd conversor_temperatura

3. Abra o arquivo **index.html** no navegador.

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos básicos de **desenvolvimento web**, como:

* Manipulação do DOM
* Eventos em JavaScript
* Estruturação de páginas HTML
* Estilização com CSS

Além disso, projetos pequenos como conversores são frequentemente usados para **treinar lógica de programação e construção de portfólio**. ([ICHI.PRO][1])

## 📸 Interface

A aplicação possui uma interface simples onde o usuário:

1. Digita um valor de temperatura
2. Escolhe a unidade de origem
3. Visualiza o resultado convertido
